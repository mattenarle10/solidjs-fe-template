# SolidJS Frontend Template

SolidJS + Vike + TailwindCSS + TanStack Query + Cognito Auth

## Setup

```bash
bun install
cp .env.example .env
bun dev
```

## Structure

```
├── components/    # UI components
├── context/       # Auth, API, Query providers
├── hooks/         # Data fetching hooks
├── lib/           # Utilities (api-client, auth, env)
├── pages/         # Vike file-based routes
├── schemas/       # Zod schemas
└── styles/        # Global CSS
```
